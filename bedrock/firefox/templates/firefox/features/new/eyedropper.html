There are a lot of reasons you might want to know the exact hex color code of a specific color — maybe you build webpages or are a graphic designer. The eyedropper tool, in the desktop version of Firefox, lets you find exact hex color codes just by hovering over any color you see on a web page.

You can find the eyedropper under “More Tools” in the Firefox toolbar menu (the far left button on your toolbar).
